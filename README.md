# MitraAri Global Analytics Dashboard 🌍

Country Demographics & Trend Prediction System

**Made by Aryan Pandey**
Live Demo:
https://mitraari-dashboard.vercel.app

## Features
- 🌍 Global Population Map
- 📊 Population Trend Analysis
- 🤖 AI Population Predictions
- 🔎 Country Insights Search
- 🌎 Country Comparison
- 💰 GDP Insights
- 🏆 Top 10 Most Populated Countries Chart
- 📋 Country Population Ranking Table

## Tech Stack
- React + Vite
- Recharts
- react-simple-maps
- REST Countries API
- World Bank API
- Python (ML population prediction)

## Project Purpose
An interactive analytics dashboard for exploring global population trends, predictions, and country-level insights.
